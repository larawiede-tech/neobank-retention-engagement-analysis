# SQL Analysis

This folder contains selected SQL queries used during the exploratory analysis, data preparation, and KPI development phases of the Neo-Bank Retention & Engagement project.

## Files

### transaction_data_quality.sql

Exploratory analysis and data quality assessment, including:

* Dataset validation
* Transaction ID uniqueness checks
* Missing value analysis
* Transaction type distribution
* Date coverage assessment
* User lifespan exploration

### mcc_exploration.sql

Merchant Category Code (MCC) enrichment and exploratory analysis, including:

* Creation and use of an MCC reference table
* MCC coverage validation
* Transaction enrichment with merchant categories
* Category-level transaction exploration

### executive_kpis.sql

Key business metrics used in the Executive Overview dashboard, including:

* Total users
* Total transactions
* Completed transaction rate
* Transaction volume
* Crypto unlock rate

These queries represent selected examples of the analytical workflow used during the project.
